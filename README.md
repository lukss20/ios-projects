iOS Applications (Swift & UIKit)

This repository contains multiple iOS applications developed using Swift and UIKit as part of academic coursework. The projects focus on user interaction, UI design, and application architecture.

Projects

Notes Application
A simple note-taking app with full CRUD functionality.

**Features:**
- Create, edit, and delete notes  
- Persistent local storage  
- Clean and responsive UI  

**Concepts:**
- UITableView  
- Data persistence  
- Navigation between screens  

 Keypad Application
An interactive numeric keypad application with real-time input handling.

**Features:**
- Dynamic input display  
- Button interactions  
- Input editing (delete/clear)  

**Concepts:**
- UIButton event handling  
- State management  
- UI responsiveness  

Contacts Application
A contact management app with multiple UI representations.

**Features:**
- TableView and CollectionView support  
- Grouped contacts with sections  
- Expand/collapse functionality  
- Swipe and gesture-based deletion  

**Concepts:**
- UITableView & UICollectionView  
- Gesture recognizers  
- Dynamic UI updates  

Technologies
- Language: Swift  
- Framework: UIKit  
- Tools: Xcode  

Skills Demonstrated
- Mobile UI Development  
- Navigation & View Controllers  
- Data Management  
- Event Handling  
- MVC Architecture  

Purpose
These projects were designed to build a strong foundation in **iOS development**, focusing on UI interaction, application structure, and user experience.
